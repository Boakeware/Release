PARExplorer.exe is a basic tool for reading DPS PAR/PVR animation
recorder hard drive archives. 

Usage:

Run ParExplorer...
Click the “...” button and select the downloaded PAR/PVR archive file.
Click “Start Scan” (the program builds an internal database for the archive, can take ~30 seconds).
Select associated PAR/PVR archive data format (you can try each option if ytou don't know which one) 
Browse or Play the image data.


Note(s):
You can use the "Static Q Factor" control to adjust for best image viewing.
